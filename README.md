# ScrapeScript
Domain email scraper that "just works." Also, converts ROT13 encoded emails, if found, to plaintext.

# Installation

```
git clone https://github.com/cybrly/ScrapeScript.git
```
```
cd ScrapeScript
```
```
python3 -m venv venv
```
```
source venv/bin/activate
```
```
python3 scriptscrape.py
```
